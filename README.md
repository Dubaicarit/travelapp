# Asia 2026 — Travel App

PWA companion app per il viaggio Seoul + Singapore + Bali.

## Come pubblicare su Vercel (5 minuti)

### Metodo 1 — Drag & Drop (più semplice)

1. Vai su https://vercel.com e crea un account gratuito
2. Dalla dashboard, clicca "Add New Project"
3. Trascina la cartella `travel-app` nella finestra del browser
4. Clicca "Deploy"
5. In 60 secondi hai l'URL della tua app (es: asia-2026.vercel.app)

### Metodo 2 — GitHub (consigliato per aggiornamenti futuri)

1. Crea account su https://github.com
2. Crea nuovo repository "asia-2026"
3. Carica i file della cartella travel-app
4. Vai su https://vercel.com → "Import Git Repository"
5. Seleziona il repository → Deploy

## Come installare sul telefono

### Android (Chrome)
1. Apri l'URL dell'app in Chrome
2. Tocca il menu ⋮ in alto a destra
3. Seleziona "Aggiungi a schermata Home"
4. Conferma — l'icona appare sulla home

### iPhone (Safari)
1. Apri l'URL in Safari (non Chrome!)
2. Tocca il tasto di condivisione (quadrato con freccia)
3. Scorri e seleziona "Aggiungi a schermata Home"
4. Conferma — l'icona appare sulla home

## Come usare la chiave API

1. Apri l'app → tab "Chiedi AI"
2. Inserisci la tua chiave API Anthropic (sk-ant-...)
3. Tocca "Salva" — viene memorizzata sul dispositivo
4. La chiave non viene mai inviata a nessun server esterno

## Funzionalità

- **Home**: riepilogo viaggio, alert prenotazioni urgenti, progresso tappe
- **Timeline**: programma giorno per giorno con checklist completamento
- **Luoghi**: tutti i POI con info pratiche, orari e prezzi
- **AI**: assistente intelligente per qualsiasi domanda sul viaggio
